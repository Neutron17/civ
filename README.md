# Neutron17's Image Viewer

## Installation

```
sudo make install
```

## Usage

```
niv [files/dirs]
```

### Navigation

go to next picture by pressing the ```right arrow key```, ```n```, ```l```, ```j```, ```s```

go to previous picture by pressing ```the left arrow key```, ```p```, ```h```, ```k```, ```w```, ```a```

Print debug info with ```d```
